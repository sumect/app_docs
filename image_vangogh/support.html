📋 常见问题解答 (FAQ)

Q: 应用无法启动或闪退怎么办？
A: 请尝试以下步骤：
1. 重启应用
2. 重启设备
3. 检查iOS系统版本是否为16.6及以上
4. 删除应用后重新安装
如问题仍存在，请联系技术支持并提供设备型号和iOS版本。
Q: 广告无法加载或显示异常？
A: 请检查：
1. 网络连接是否正常
2. 重启应用重新加载广告
3. 确保设备有足够的存储空间
4. 考虑购买高级版本以移除所有广告
注：我们使用非个性化广告，无需追踪权限
Q: 激励广告观看后未解锁内容？
A: 请确保：
1. 完整观看广告直到结束
2. 网络连接稳定
3. 不要在广告播放期间切换应用
如仍有问题，请重启应用或联系技术支持。
Q: 内购无法完成或恢复？
A: 请尝试：
1. 确认Apple ID账户正常
2. 检查支付方式是否有效
3. 在应用设置中点击"恢复购买"
4. 重启应用后再次尝试
如需退款，请联系Apple客服。
Q: 图片加载缓慢或失败？
A: 建议：
1. 检查网络连接
2. 清理设备存储空间
3. 重启应用
4. 在WiFi环境下使用以获得最佳体验
Q: 应用是否会追踪我的个人信息？
A: 不会。我们的隐私承诺：
1. 不收集任何个人信息
2. 使用非个性化广告，无需追踪权限
3. 所有数据仅存储在您的设备本地
4. 购买高级版本后完全移除广告
详情请查看我们的隐私政策

🛠️ Troubleshooting Guide

App Performance Issues

  • Ensure device has sufficient storage space (recommend at least 1GB)
  • Close other memory-intensive apps
  • Restart device regularly to clear memory
  • Update to latest app and iOS versions

Network-Related Issues

  • Check WiFi or cellular data connection
  • Try switching network environments
  • Check if VPN or proxy is enabled
  • Reset network settings (Settings → General → Transfer or Reset iPhone → Reset → Reset Network Settings)
